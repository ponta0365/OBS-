=========================================
      OBS字幕メーカー 使い方ガイド
=========================================

【はじめて使うとき】
1. このフォルダの中にある「setup.bat」をダブルクリックして実行します。
2. 黒い画面が開き、必要なプログラムのインストールが始まります。
3. 「インストールが完了しました！」と表示されたら画面を閉じます。
※この作業は最初の1回だけでOKです。

【アプリの起動】
1. 「run.bat」をダブルクリックします。
2. アプリの画面が開きます。

【使い方】
1. OBS Studioを起動し、WebSocketサーバーを有効にしておきます。
   （ツール -> WebSocketサーバー設定 -> 認証を有効にする）
2. アプリ画面でOBSのパスワードを入力し、「Fetch OBS Lists」を押します。
3. 録画に使用したい「プロファイル」と「シーンコレクション」を選びます。
4. 「設定字幕（Ctrl+Alt+Tで入力）」や「字幕表示秒数（秒）」を入力します。
5. 「Save Settings」を押して設定を保存します。
6. 「Start Recording」を押すか、設定した録画開始/停止ホットキー（初期値: Ctrl + Alt + R）を押すと録画が開始されます！
7. 録画中は、設定したホットキーで字幕やチャプターを記録できます：
   ・字幕記録（初期値: Ctrl + Alt + T）: 事前入力した「設定字幕」をタイムスタンプ付きで記録。
   ・入力小窓（初期値: Alt + G）       : 自由入力窓を表示。入力開始時のタイムスタンプで字幕が記録され、Enterで自動クリア＆連続入力が可能。
   ・自動チャプター（初期値: Alt + C）: 連番チャプター（チャプター 1, 2...）を自動カウントアップで記録。
8. もう一度録画開始/停止ホットキー（初期値: Ctrl + Alt + R）を押すと録画が停止し、動画と同じ場所に字幕ファイル（.srt）が作成されます！

【ショートカットキーが動作しない場合の対策】
- 他のアプリやゲームとショートカットキーが競合している場合、アプリ画面で別のキーを設定し「Save Settings」を押してください。
- ゲームやOBS等、管理者権限で実行されている別アプリが最前面にある場合、本アプリも「管理者として実行」で起動しないとショートカットキーが反応しません。
- キー監視が一時的に外れて反応しなくなった場合は、画面左側の「Reset Hotkeys」ボタンを押してフックを再登録してください。
