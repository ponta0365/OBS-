=========================================
      OBS字幕メーカー 使い方ガイド
=========================================

【はじめて使うとき】
1. このフォルダの中にある「setup.bat」をダブルクリックして実行します。
2. 黒い画面が開き、必要なプログラムのインストールが始まります。
3. 「インストールが完了しました！」と表示されたら画面を閉じます。
※この作業は最初の1回だけでOKです。

【アプリの起動】
1. 「run.bat」をダブルクリックします。
2. アプリの画面が開きます。

【使い方】
1. OBS Studioを起動し、WebSocketサーバーを有効にしておきます。
   （ツール -> WebSocketサーバー設定 -> 認証を有効にする）
2. アプリ画面でOBSのパスワードを入力し、「Fetch OBS Lists」を押します。
3. 録画に使用したい「プロファイル」と「シーンコレクション」を選びます。
4. 「設定字幕（Ctrl+Alt+Tで入力）」や「字幕表示秒数（秒）」を入力します。
5. 「Save Settings」を押して設定を保存します。
6. 「Start Recording」を押すか、キーボードの『Ctrl + Alt + R』を押すと録画が開始されます！
7. 録画中は、以下のショートカットキーで字幕やチャプターを記録できます：
   ・『Ctrl + Alt + T』: 設定した「設定字幕」を記録します。
   ・『Alt + G』       : 自由入力用の「小窓」を表示します。テキストを入力して『Enter』を押すと、入力開始時のタイムスタンプで字幕が記録され、入力欄がクリアされます（開いたまま連続入力が可能です）。
   ・『Alt + C』       : 「チャプター 1」「チャプター 2」といった連番チャプターを自動記録します。
8. もう一度『Ctrl + Alt + R』を押すと録画が停止し、動画と同じ場所に字幕ファイル（.srt）が作成されます！
