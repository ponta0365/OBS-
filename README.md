# OBS字幕メーカー (OBS Subtitle Maker)

OBS Studioでの録画中に、ショートカットキーを押すだけでリアルタイムに字幕（SRT形式）やチャプターマーカーを生成できるWindows向けデスクトップアプリケーションです。

---

## 🌟 主な機能
- **録画と連動**: アプリ側からOBSの録画開始・停止を制御（キーボードショートカット `Ctrl + Alt + R` にも対応）。
- **リアルタイム字幕入力**: 録画中に `Ctrl + Alt + T` であらかじめ設定した定型字幕をタイムスタンプ付きで記録。
- **自由入力小窓**: `Alt + G` で入力用の小窓を表示。テキストを入力して `Enter` を押すと、入力した時点のタイムスタンプで字幕が即座に記録されます。
- **自動チャプター作成**: `Alt + C` を押すごとに「チャプター 1」「チャプター 2」と自動カウントアップされるチャプターマーカーを記録。
- **日付別フォルダ自動作成**: 基準フォルダを指定し、「Create Date Folder」ボタンを押すことで、今日の日付（例: `YYYY-MM-DD`）のフォルダを自動作成し、OBS側の保存先ディレクトリにリアルタイム反映します。
- **SRT自動書き出し**: 録画終了時、録画した動画ファイルと全く同じフォルダに、同名の `.srt` 字幕ファイルを自動で生成・保存します。

---

## ⚙️ 動作要件
- OS: Windows 10 / 11
- OBS Studio (WebSocket サーバーが有効になっていること)
- Python 3.10 以上 (ソースコードから実行する場合)

---

## 🚀 使い方（配布パッケージ版）

1. [Releases ページ](https://github.com/ponta0365/OBS-/releases) から最新の `OBS字幕メーカー.zip` をダウンロードして解凍します。
2. フォルダ内の `setup.bat` をダブルクリックして実行し、必要な環境を自動セットアップします（最初の1回のみ）。
3. セットアップ完了後、`run.bat` をダブルクリックしてアプリを起動します。
4. OBS Studio を起動し、「ツール」->「WebSocketサーバー設定」からサーバーを有効にします。
5. アプリの画面でOBSの接続パスワードを入力し、「Fetch OBS Lists」を押してプロファイル等のリストを取得・設定します。
6. 後は画面の指示に従って録画を開始するだけです！
